GM Drinks - cardápio

Job

Todas as fases (Full)

Solicitante

GM Drinks

GM Drinks é um espaço onde o foco é servir drinks com excelência.

Neste projeto de cardápio desenvolvi, com base nas referências que recebi do solicitante, um layout intuitivo e que transmitisse uma atmosfera medieval- que é a temática do bar. A tipografia, as cores e os elementos selecionei a partir dessa premissa e juntos formaram um cardápio uniforme e agradável.

Este site possui somente a versão mobile uma vez que o intuito do solicitante era disponibila-lo através de um QR Code.

GM Drinks foi um projeto enxuto, porém conseguimos criar um visual interessante e sólido.

O site está disponível aqui.
